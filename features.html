  • The game

    Mini-Madness is a single player 2d car-racing game. It's got one track, one car and three laps. The player controls the car with arrow-keys. Left and right for turning, up-arrow for acceleration and down-arrow will slam the breaks! Watch out for those curbs, you might get stuck on them and loose valuable time...

  • Features

    Mini-Madness is suitable to run on any JavaScript/HTML5 compatible platform. The game is presented to the user in a very straight forward and simple form, without any extra settings or menu choices. Therefore it is also very suitable to add to any website and easy to visually package it to fit into the existing HTML/CSS design.
